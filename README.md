# bank-churn-analysis
Exploratory analysis of bank customer churn dataset

## Dataset
Source: Kaggle Bank Customer Churn Dataset  
Link: https://www.kaggle.com/datasets/radheshyamkollipara/bank-customer-churn/data

## Objective
The goal of this analysis is to understand customer churn patterns and prepare the data for supervised machine learning models focused on churn prediction and financial impact.
