Dataset
Source: Kaggle Bank Customer Churn Dataset

Link: https://www.kaggle.com/datasets/radheshyamkollipara/bank-customer-churn/data

Dataset Details:

10,000 customer records
18 features (demographics, account details, engagement metrics)
Target: Customer churn (20.4% churn rate)
Objective
The goal of this analysis is to understand customer churn patterns and prepare the data for supervised machine learning models focused on churn prediction and financial impact.

Research Questions:

Which customer attributes most strongly drive churn risk?
How does incorporating financial exposure alter prioritization?
What is the optimal intervention threshold?
